# cfd1 — Relatórios e simulações de CFD (Trabalhos 1 e 2)

Este repositório publica, via GitHub Pages, páginas interativas dos trabalhos de CFD (UFPR) com simulações e gráficos executados no navegador. A proposta é oferecer um ponto único de acesso para:

- Visualizar o Trabalho 1 e o Trabalho 2 diretamente no browser
- Executar e ajustar parâmetros das simulações sem instalar nada
- Consultar o código-fonte correspondente

## Acesso rápido

- Home: [rbeninca.github.io/cfd1](https://rbeninca.github.io/cfd1/)
  - Trabalho 1: [trabalho1](https://rbeninca.github.io/cfd1/trabalho1/)
  - Trabalho 2: [trabalho2](https://rbeninca.github.io/cfd1/trabalho2/)
- Repositório: [github.com/rbeninca/cfd1](https://github.com/rbeninca/cfd1)

## Estrutura

- `index.html` — página índice que direciona para os trabalhos
- `trabalho1/index.html` — relatório e simulação do Trabalho 1
- `trabalho2/index.html` — relatório e simulação do Trabalho 2



## Tecnologias
- JavaScript no navegador (sem build)
- Chart.js (gráficos) e MathJax (fórmulas)
